Gateways de Pagamento para Focatto

Entenda as melhores opções para receber pagamentos no Brasil (Pix, Cartão de Crédito e Boleto), simule taxas de repasse (Split) e saiba o que é necessário para se cadastrar.

ℹ
O que é Split de Pagamento? Como a Focatto conecta clientes a profissionais (Luthiers e Professores), o ideal é usar um gateway com suporte a split. Isso garante que, quando um cliente paga, o valor é dividido automaticamente: a taxa da plataforma (comissão Focatto) vai para sua conta bancária e o restante vai direto para a conta do profissional, evitando bitributação e reduzindo burocracia.

Como Funciona a Integração Técnica e o Split?

1

Cliente faz a compra no site

O cliente escolhe um serviço de luthier ou contrata um professor no Focatto e realiza o pagamento via Pix ou Cartão.

2

Gateway processa e aplica o Split

A API do Gateway intercepta o pagamento. Com base na configuração do Split que enviamos no backend, ele separa a comissão da plataforma imediatamente.

3

Liquidação direta e segurança tributária

O prestador (luthier/professor) recebe a parte dele diretamente na subconta dele do Gateway, e a Focatto recebe a comissão. A emissão de notas fiscais pode ser configurada para ser automática para cada parte, eliminando riscos de bitributação.

Veja como é simples criar um pagamento contendo split usando a API do Asaas no Node.js/Next.js. Você informa a comissão do parceiro ou o valor fixo a ser transferido:

const axios = require('axios');

// Exemplo de chamada na API do Asaas para criar uma cobrança com split
async function criarCobrancaComSplit() {
    const url = 'https://api.asaas.com/v3/payments';
    const token = 'Sua_Chave_De_API_Gerada_No_Painel_Do_Asaas';

    const payload = {
        // Dados da cobrança do Cliente
        "customer": "cus_000005747492", // ID do cliente previamente cadastrado
        "billingType": "CREDIT_CARD",   // CREDIT_CARD, PIX ou BOLETO
        "value": 300.00,                 // R$ 300,00
        "dueDate": "2026-06-15",

        // Regra de divisão do dinheiro (Split)
        "split": [
            {
                "walletId": "d48509ca-85f0-4fa2-bf66-d44b1c8c8230", // ID da carteira do Luthier/Professor
                "percentualValue": 90                                // Fica com 90% (R$ 270)
            }
            // O restante (10%) fica automaticamente na carteira principal da Focatto
        ]
    };

    try {
        const response = await axios.post(url, payload, {
            "headers": {
                "access_token": token
            }
        });
        console.log('Cobrança com Split criada com sucesso!', response.data);
    } catch (error) {
        console.error('Erro ao criar split:', error.response?.data || error.message);
    }
}

Se você receber os R$ 300,00 na conta bancária da Focatto e depois transferir R$ 270,00 para o luthier de forma manual:

  • A Receita Federal entenderá que a Focatto faturou R$ 300,00 inteiros e você terá que pagar imposto (ex. 6% a 15%) sobre o valor total, mesmo repassando a maior parte.
  • O luthier também terá que emitir uma nota e pagar imposto sobre os R$ 270,00 dele. Isso se chama bitributação.

Como o Split resolve isso: Como o dinheiro é dividido antes de entrar no banco (no próprio gateway), a Focatto declara apenas a comissão (R$ 30,00) e o luthier declara os R$ 270,00. Isso é legalmente respaldado e economiza milhares de reais em impostos no final do mês.
